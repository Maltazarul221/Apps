# Apps

A collection of useful web applications.

## Meeting Notes App - Enhanced Version

A powerful, feature-rich web application for taking professional notes during meetings with calendar synchronization.

### Key Features

#### Core Functionality
- Create and manage meetings with full details (title, dates, attendees, location, tags)
- Add timestamped notes during meetings
- Mark important notes as action items with special highlighting
- Edit existing notes with visual feedback
- Auto-save to browser local storage
- Delete individual notes or entire meetings

#### Advanced Features
- **Dark Mode**: Beautiful dark theme with persistent preference
- **Search**: Full-text search across meetings, notes, and tags
- **Filtering**: Filter by date range (today/week/month) or action items
- **Sorting**: Multiple sort options (recent, oldest, title, note count)
- **Statistics**: Comprehensive dashboard showing meeting and note analytics
- **Tags System**: Organize meetings with customizable tags
- **Meeting Duration**: Track start and end times
- **Location/Links**: Add meeting rooms or video call URLs

#### Calendar Synchronization
- **Export to .ICS**: Export meetings as iCalendar files for Google Calendar, Outlook, Apple Calendar, etc.
- **Import from .ICS**: Import calendar events and convert them to meeting notes
- **Text Export**: Download meetings as formatted text files

#### UI/UX
- Responsive design for desktop, tablet, and mobile
- Smooth animations and transitions
- Custom scrollbars
- Modal dialogs
- Beautiful gradient design
- Hover effects and visual feedback

### Quick Start
1. Navigate to the `meeting-notes` directory
2. Open `index.html` in your web browser
3. Start taking notes!
