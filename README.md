# Apps

A collection of useful web applications.

## Meeting Notes App

A simple, elegant web application for taking notes during meetings.

### Features
- Create and manage meetings with titles, dates, and attendees
- Add timestamped notes during meetings
- Mark important notes as action items
- Auto-save to browser local storage
- Export meeting notes as text files
- Beautiful, responsive design

### Quick Start
1. Navigate to the `meeting-notes` directory
2. Open `index.html` in your web browser
3. Start taking notes!

For detailed instructions, see [meeting-notes/README.md](meeting-notes/README.md)
